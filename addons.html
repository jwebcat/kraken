Add-Ons with Some Sass

Kraken-Sass is essentially just a complete conversion of Kraken into Scss. Everything that will work in the normal Kraken, will work here too.

Kraken was built to be lightweight but expandable. Add functionality with add-ons.

Mobile & Responsive Web

  • Icon fonts, a lightweight, scalable way to include icons.
  • FluidVids.js, a lightweight javascript snippet for fluid iframes.
  • .htaccess, best practice server rules for making web pages fast and secure.

Interactive Content (Sass Versions Coming Soon...)

  • Astro, a collection of mobile-first navigation patterns.
  • Drop, mobile-friendly dropdown menus.
  • Houdini, a simple expand-and-collapse widget.
  • Tabby, lightweight, mobile-first toggle tabs.
  • Modals, simple modal dialogue windows.
  • Slider, a responsive, touch-enabled image carousel.
  • SmoothScroll.js, simple jQuery script to animate scrolling to anchor links.

CSS Components converted to Scss

  • Snapshot Sass, basic image styling.
  • Tables Sass, simple CSS table styling.
  • Hero Sass, basic callout boxes and background colors.
  • Social Sharing links and buttons without the bloat.
  • Bar Graph Sass, lightweight CSS bar graphs.
  • Alerts Sass, simple alert messages.
  • Spinners Sass, CSS rotation animations.

WordPress

  • Kraken for WordPress, a WordPress theme boilerplate.
  • WordPress for Web Apps, transform WordPress into a web app engine.
  • WordPress Shopping Cart, a simple PayPal shopping cart.
  • Petfinder for WordPress, functions for working with the Petfinder API.
